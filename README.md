# My React Blog - HYEHYE
이 프로젝트는 리액트(React)로 개발한 개인 블로그입니다.

## 환경
- Node.js v20.14.0+
- Yarn v4.3.1+
- vite v5.3.1+

## 기술
- React
- JavaScript
- Tailwind CSS
- Zustand
- Axios
- MSW

## 실행 방법
1. 레파지토리 클론:
```
git clone <repository-url>
cd <project-directory>
```

2. 의존성 설치:
```
yarn install
```

3. 개발 서버 시작:
```
yarn dev
```
